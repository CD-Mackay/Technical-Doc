FreeCodeCamp

FreeCodeCamp should be the first stop on the road to success for an ambitious new developer. This open-source learning resource provides students with lessons beginning from total basics, all the way up to complex subjects such as machine learning

As a testament to FCC's efficacy, the author of this page had no prior coding experience three months ago

Codewars

Codewars is less of a stand-alone resource than FCC, but nonetheless can be extremely useful in developing coding skills. Many issues in web development require Devs to calmly work towards logical solutions to issues. Codewars provides practice problems for Devs to sharpen their skills without the hassle of opening existing projects

HTML

HTML stands for Hypertext Markup Language, and is one of the primary building blocks of the modern internet. See below for an example of how it is used to create web pages

#introduction{ text-size: 20px; }

The example above changes the font size of the introduction paragraph from the standard font to the user-selected 20 pixels

CSS

CSS, or Cascading Style Sheets provides instructions for the aesthetic aspects of web design. Formatting of elements, color schemes, text fonts and many other aspects of design are controlled by CSS code

#introduction{ color: black; background-color: gray; }

The script above provides styling instructions for an element classified as the introduction. According to the code above the introductory section of the document will contain black text over a gray background

CSS can also be used to alter the physical layout of a web-page, displaying HTML content in multiple formats depending on the device.
@media only screen and (max-width: 600px) { body { background-color: lightblue; } }

The code above is known as a media query. For devices with a screen-width of 600 pixels or less, the background will be displayed as light blue.
@media only screen and (min-width: 600px) { body { background-color: black; } }

This second media query addresses larger screen sizes. Notice that the code only comes into effect if the screen size is larger than 600 pixels.
